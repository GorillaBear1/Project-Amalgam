===================================
   S O V E R E I G N   C O D E X
===================================

[ NAVIGATION ]
  cd [folder]   :: Go to folder
  cd ..         :: Go back/up
  ls            :: List files

[ CREATION ]
  mkdir [name]  :: Make new folder
  touch [file]  :: Make empty file
  nano [file]   :: Edit file (Ctrl+O Save, Ctrl+X Exit)

[ TIMELINE (GIT) ]
  git status    :: Check changes
  git add .     :: Stage changes
  git commit -m :: Save changes
  git push      :: Upload to Cloud

[ SYSTEM ]
  python3 nexus.py :: Launch App
  pip install [x]  :: Install tools
  exit             :: Close Terminal
